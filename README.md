# Ativify — Software de Gestão de Ativos

Sistema desktop para gerenciamento de ativos patrimoniais, desenvolvido durante o 2º bimestre do 3º ano do Ensino Médio Técnico Integrado com TI no Senac.

> **Projeto acadêmico** em Java Swing com interface gráfica completa

---

## Tecnologias

| Categoria | Stack |
|-----------|-------|
| Linguagem | [Java 24](https://jdk.java.net/24/) |
| Interface | [Java Swing](https://docs.oracle.com/javase/tutorial/uiswing/) |
| Build | [Apache Ant](https://ant.apache.org/) |
| IDE | [NetBeans](https://netbeans.apache.org/) |

---

## Funcionalidades

- **Login** com autenticação de usuário e senha
- **Dashboard** com navegação centralizada para todos os módulos
- **Cadastro de ativos** com formulário completo de registro
- **Listagem de ativos** com busca, edição e exclusão
- **Depreciação patrimonial** com cálculo e exibição em tabela
- **Categorias** para organização dos ativos
- **Gestão de funcionários** com cadastro e listagem
- **Relatórios** gerenciais do sistema
- **Configurações** e preferências do sistema
- **Ajuda** com informações sobre o Ativify

---

## Rodando localmente

```bash
git clone https://github.com/Raggzinn/ativify.git
cd ativify
```

**Pelo NetBeans:** abra o projeto com **File > Open Project** e rode com **F6**

**Pelo terminal:**

```bash
ant run
```

> Requer [JDK 24](https://jdk.java.net/24/) ou superior
